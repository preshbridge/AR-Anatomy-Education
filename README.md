## 🧠 Project Overview

This project is an interactive mobile application designed to aid Nigerian students in learning human anatomy. The core functionalities include a 3D interactive model, a searchable anatomical dictionary, and a quiz module for self-assessment. This document outlines the project's final implementation, testing strategies, and results.
