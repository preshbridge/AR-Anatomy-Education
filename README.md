# AR Anatomy Education

## Overview

AR Anatomy Education is an Augmented Reality (AR) mobile application developed to enhance anatomy learning through interactive 3D visualisation. The application allows students to study selected muscles of the upper limb by reading educational content and viewing the corresponding muscle in Augmented Reality.

The project was developed as part of the Bachelor of Software Engineering Capstone Project at the African Leadership University.

---

## Features

* Interactive study modules for upper limb muscles
* Educational lesson pages for each muscle
* Augmented Reality visualisation of muscles
* Organised muscle categories:

  * Shoulder Muscles
  * Upper Arm Muscles
  * Forearm Muscles
* Easy-to-use navigation
* Modern user interface

---

## Installation Guide

### Requirements

* Android device with ARCore support
* Android 8.0 or later
* Camera permission enabled

### Steps

1. Download the APK file.
2. Install the application on your Android device.
3. Grant camera permission when prompted.
4. Open the application.
5. Signup/login
6. Navigate to **Study**.
7. Select a muscle group.
8. Choose a muscle.
9. Read the lesson.
10. Tap **View in AR**.
11. Point the camera at a flat surface and tap to place the model.

---

## How to Run the Project

1. Clone the repository.

```bash
git clone  https://github.com/preshbridge/AR-Anatomy-Education/tree/main

```

2. Open the project using Unity 2022.3.30f1.

3. Ensure AR Foundation and ARCore packages are installed.

4. Build and run on an Android device.
